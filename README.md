# Deloitte Australia Data Analytics Job Simulation

## Overview
Completed Deloitte Australia's Data Analytics Job Simulation on Forage (May 2026),
working on real-world data problems across two tasks involving Tableau dashboards
and Excel-based data classification.

## Tools Used
- Tableau
- Microsoft Excel
- Dataset: Daikibo Industries Telemetry Data (JSON)

## Task 1 — Factory Downtime Analysis (Tableau)
Analysed telemetry data from 4 global factories to identify machine downtime patterns.

### Key Insights
- Daikibo Factory Seiko (Osaka, Japan) recorded the highest downtime
- LaserWelder was responsible for ~90% of Seiko's total downtime
- Daikibo Berlin recorded the lowest downtime across all factories

## Task 2 — Equality Classification (Excel)
Classified employee equality scores across 37 job roles using nested IF formulas.

### Classification Logic
- Fair: Score between -10 and +10
- Unfair: Score between -20 and -11 or +11 and +20
- Highly Discriminative: Score below -20 or above +20
